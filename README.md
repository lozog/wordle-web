# Wordle Clone for web

A clone of [Wordle](https://www.powerlanguage.co.uk/wordle/).

Built using React, hosted with Netlify.

## Run
```
yarn && yarn start
```
or
```
yarn && netlify dev
```

## Build & Deploy
```
yarn build
netlify deploy --prod
```
